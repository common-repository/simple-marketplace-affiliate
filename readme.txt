Binde mit Shortcodes beliebige Marketplace Produkte auf deiner WordPress Seite ein und setze in den Optionen deinen Affiliate Tag, um auch noch Geld damit zu verdienen.

== Description ==

Mit dem Simple Marketplace Affiliate WordPress Plugin ist es kinderleicht Envato Market Produkte, auf deiner Webseite zu pr&auml;sentieren. Da das Plugin mit der Envato API zusammenarbeitet, werden immer aktuelle Daten per jQuery AJAX abgerufen. Es gibt also keine langen Ladezeiten, beim Aufrufen deiner Website. 

In den Plugin Optionen kannst du deinen Affiliate-ID und deinen pers&ouml;nlichen Token setzen. AuÃŸerdem gibt es zwei verschieden Shortcodes. Du kannst einen w&auml;hlen und f&uuml;r Dich selbst entscheiden, wie Du Envato Produkte auf Deiner Website ver&ouml;ffentlichst.

**Produkt-ID**  

Die Produkt-ID findest du in der URL der jeweiligen Produktseiten. Hier im Beispiel lautet die Produkt-ID: 15196030

`XYZ.net/item/onyx-responsive-wordpress-blog-theme/15196030`

**Shortcode**  

`[envato id="XYZ"]`

**Alternativer Shortcode**  

`[eawp id="XYZ"]`

== Installation ==

1. Lade die Plugindateien in Dein `/wp-content/plugins/plugin-name` Verzeichnis, oder installiere das Plugin mit Hilfe der **WordPress -> Plugins** Seite in WordPress.
1. Aktiviere das Plugin auf der Plugin Seite in WordPress.
1. Richte das Plugin in den Plugin Einstellungen unter `Simple Marketplace Affiliate` ein.
1. Wichtig ist, dass Du f&uml;r das Plugin einen pers&ouml;nlichen Token der Envato API ben&ouml;tigst. Weitere Informationen dazu, findest Du in den Plugin Einstellungen.
